Side-by-Side Positioning Demo

These two divs are positioned next to one another using display: inline-block

Notice the space between them. It occurs because of my carriage return in the HTML which I've used to keep the code organised. I've used reset to make insure no elements should have any default margins or padding.(I did add a bottom margin to make ot easier to read.) this space is actually a space character rendered just as it would be between two inline elements.

These two positioned usingfloat: left.

You can see that, even though the elements are spaced in the code, the space is not actuall rendered in the page. This may not be a huge deal in these first examples, but...

what if I want my side-by-side element to take up the full width of the parent element? I can set the width of each to 50%, right?

Wrong! because, together,the two inline-block elements will add up to 100% of the available width, but just does'nt account for the space between.

If I do the same thing with floated elements...

The two elements are side-by-side just as I intended and fill the entire width of the parent(which, in this case, is the same as the width of the viewport.)
